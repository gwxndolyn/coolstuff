# coolstuff

A modern **React + TypeScript + Tailwind CSS + Vite** starter template.

## Stack

| Tool | Version |
|------|---------|
| [Vite](https://vite.dev/) | v8 |
| [React](https://react.dev/) | v19 |
| [TypeScript](https://www.typescriptlang.org/) | v5 |
| [Tailwind CSS](https://tailwindcss.com/) | v4 |

## Features

- ⚡ Blazing-fast HMR with Vite
- 🎨 Utility-first styling with Tailwind CSS v4
- 🔒 Fully type-safe with TypeScript
- 📱 Responsive layout out of the box
- 🧩 Component-based architecture

## Template Pages / Sections

- **Navbar** – sticky top nav with mobile hamburger menu
- **Hero** – gradient hero with CTA buttons
- **Features** – 6-card feature grid
- **Pricing** – 3-tier pricing table with highlighted plan
- **CTA** – full-width call-to-action banner
- **Footer** – multi-column footer with links

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview
```
